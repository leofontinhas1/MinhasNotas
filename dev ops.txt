para derrubar módulo e subir de novo 

entrar na maquina maneger 1

enrtar no diretório /home/ec2-user/docker/docker_apptools

	cd /home/ec2-user/docker/docker_apptools

executar satck_rm.sh para derrubar o módulo

	sh stack_rm.sh

em seguida executar stack_deploy.sh para subir módulo

	sh stack_deploy.sh

quando o módulo estiver com algun problema para subir que seja parecido com isso :

	2020-01-16 09:04:43 INFO  liquibase:92 - Waiting for changelog lock...

rodar o seguinte comando no banco de dados desse módulo

	UPDATE DATABASECHANGELOGLOCK SET LOCKED=0, LOCKGRANTED=null, LOCKEDBY=null where ID=1;
	commit;
